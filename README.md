# Sistema Bancário em Python - Versão 1 (Ainda não dominei funções/def

Projeto desenvolvido em Python para praticar lógica de programação, estruturas condicionais, laços de repetição e tratamento de erros.

## Funcionalidades

- Consultar saldo
- Realizar depósitos
- Realizar saques
- Validação de valores inválidos
- Verificação de saldo insuficiente
- Confirmação de operações
- Tratamento de erros com try/except
- Menu interativo no terminal

## Tecnologias utilizadas

- Python

## Conceitos praticados

- Variáveis
- if / elif / else
- while
- try / except
- Validação de entrada
- Operadores lógicos
- Manipulação de strings
- Fluxo de execução
